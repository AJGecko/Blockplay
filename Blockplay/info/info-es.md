# Blockplay - Info

Versión actual: 0.1.0 

---
## Cómo jugar

- Mover: W, A, S, D o flechas
- Saltar: Espacio, W o flecha arriba
- Caída rápida: S o flecha abajo
- Pausa: Escape o P
- También puedes controlar al jugador con el ratón. (Haz clic y mantén para moverte, suelta para detenerte, haz clic encima del jugador para saltar, haz clic debajo del jugador para caída rápida). Esto también funciona con pantalla táctil.

> Muchas funciones también se pueden controlar mediante botones en pantalla.

## Sobre la dificultad

- Normal: Puedes aterrizar desde arriba en la plataforma, pero caerás si aterrizas desde el lado o desde abajo.
- Fácil: Basta con tocar la plataforma para subirte a ella.
